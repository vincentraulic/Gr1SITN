PMO-web (Project Management Online)
=============

Software of project management

Technos/Frameworks utilisés pour le projet
-----------
- JSF
- EJB
- CDI
- Spring security 4
- Serveur Glassfish 4
- Base de données MySQL

Branches
-----------
- [projet en jsf](https://github.com/vincentraulic/Gr1SITN/tree/dev_jsf)
- [projet en Spring MVC](https://github.com/vincentraulic/Gr1SITN/tree/dev) Version obsolete

Installation
-----------

```
Eclipse :

1) Importer le projet maven

Base de données :

1) Exécuter le script script_database.sql

Connexion du serveur Glassfish à la base de données

1) La data-source située dans le fichier persistence.xml s'appelle "jdbc/pmo_resource"
```
